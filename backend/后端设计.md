# 数据库设计

### 1. users (用户表)
| 字段名          | 数据类型         | 约束/默认值 | 说明                        |
| :----------- | :----------- | :----- | :------------------------ |
| `id`         | INT UNSIGNED |        | 用户唯一标识                    |
| `openid`     | VARCHAR(64)  |        | 微信OpenID，用于登录绑定           |
| `nickname`   | VARCHAR(50)  |        | 用户昵称                      |
| `avatar_url` | VARCHAR(500) |        | 头像URL                     |
| `role`       | ENUM         |        | 角色：user(普通用户), admin(管理员) |
| `created_at` | DATETIME     |        | 注册时间                      |
| `updated_at` | DATETIME     |        | 最后更新时间                    |

---

### 2. flowers (花卉表)
| 字段名                      | 数据类型         | 约束/默认值 | 说明                                                         |
| :----------------------- | :----------- | :----- | :--------------------------------------------------------- |
| `id`                     | INT UNSIGNED |        | 花卉ID                                                       |
| `species`                | VARCHAR(100) |        | 花卉品种名称（如：樱花）                                               |
| `scientific_name`        | VARCHAR(200) |        | 学名                                                         |
| `family`                 | VARCHAR(100) |        | 所属科                                                        |
| `genus`                  | VARCHAR(100) |        | 所属属                                                        |
| `bloom_status`           | ENUM         |        | 花期状态：dormant(休眠), budding(含苞), blooming(盛开), withering(凋谢) |
| `historical_bloom_start` | VARCHAR(20)  |        | 历史平均始花期（如：03-01）                                           |
| `historical_bloom_end`   | VARCHAR(20)  |        | 历史平均末花期（如：03-20）                                           |
| `cover_image`            | VARCHAR(500) |        | 封面图URL                                                     |
| `description`            | TEXT         |        | 花卉详细描述                                                     |


---

### 3. places (地点表)
| 字段名           | 数据类型          | 约束/默认值 | 说明   |

---

### 5. checkins (打卡记录表)
| 字段名               | 数据类型         | 约束/默认值 | 说明                                  |
| :---------------- | :----------- | :----- | :---------------------------------- |
| `id`              | INT UNSIGNED |        | 打卡ID                                |
| `user_id`         | INT UNSIGNED |        | 打卡用户ID                              |
| `flower_place_id` | INT UNSIGNED |        | 打卡的具体花卉地点ID                         |
| `bloom_report`    | ENUM         |        | 用户目击状态：budding, blooming, withering |
| `content`         | TEXT         |        | 打卡文字内容（限500字）                       |
| `images`          | JSON         |        | 图片对象数组（三级图片URL）                     |
| `likes_count`     | INT UNSIGNED |        | 点赞数                                 |
| `created_at`      | DATETIME     |        | 打卡时间                                |

# 需要的API

## 一、用户相关API（已有，略作整理）

| 路径                   | 说明                |
| -------------------- | ----------------- |
| `/api/user/register` | 注册                |
| `/api/user/login`    | 登录（微信OpenID）      |
| `/api/user/info`     | 修改用户信息（昵称、头像等）    |
| `/api/user/info`     | 获取当前用户信息（含徽章、称号等） |


## 二、花卉相关API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/flowers` | 获取花卉列表（支持按状态、种类筛选） |
| GET | `/api/flowers/:id` | 获取单个花卉详细信息（含地点列表） |
| GET | `/api/flowers/:id/bloom-status` | 获取花卉当前盛开状态（实时/平均） |

## 三、地点与地图相关API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/places` | 获取地点列表（支持按花卉种类筛选） |
| GET | `/api/places/:id` | 获取地点详细信息（含该地花卉信息） |
| GET | `/api/map/flowers` | 地图模式：返回所有花卉地点及状态（用于地图标注） |
| GET | `/api/map/filter` | 地图筛选：按花卉种类返回地点坐标 |

---

## 四、打卡/帖子相关API

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/checkins` | 打卡提交（图片、文字、状态、flower_place_id、时间） |
| GET | `/api/checkins` | 获取打卡列表（支持按时间、状态、位置筛选） |
| GET | `/api/checkins/:id` | 获取单个打卡详情（含用户信息、花卉、地点） |
| PUT | `/api/checkins/:id/like` | 点赞/取消点赞 |
| GET | `/api/flowers/:id/checkins` | 获取某花卉下的所有打卡帖子（按时间排序） |
| GET | `/api/places/:id/checkins` | 获取某地点下的所有打卡帖子 |

---

## 五、成就与称号相关API

| 方法  | 路径                        | 说明                 |
| --- | ------------------------- | ------------------ |
| GET | `/api/achievements`       | 获取所有成就列表           |
| GET | `/api/user/achievements`  | 获取当前用户已获得的成就       |
| GET | `/api/user/titles`        | 获取当前用户已获得的称号       |
